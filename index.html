How do I install wrkbnch?
Run npm install wrkbnch-css and link the stylesheet in your HTML head.
Does it support dark mode?
Yes. Use data-mode="dark" on the html element, or omit it for automatic system preference.
Can I use it with React/Vue?
Absolutely. wrkbnch is pure CSS, so it works with any framework or none at all.

FAQ List

Does wrkbnch need JavaScript to render components?

No. Components are pure CSS and semantic HTML, so they drop into static pages, React apps, and server-rendered tools without a client runtime.

Can I swap in brand tokens for a client workspace?

Yes. The system is token-driven, so you can tune accent, neutrals, and radius decisions without rewriting component structure.

Is it suitable for both docs and internal dashboards?

That is the sweet spot. wrkbnch is designed for product surfaces that mix editorial tone with dense operational data.
