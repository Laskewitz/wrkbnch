How do I install wrkbnch?
Run npm install wrkbnch-css and link the stylesheet in your HTML head.
Does it support dark mode?
Yes. Use data-mode="dark" on the html element, or omit it for automatic system preference.
Can I use it with React/Vue?
Absolutely. wrkbnch is pure CSS — it works with any framework or none at all.
